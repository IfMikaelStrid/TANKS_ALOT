#TANKS
OS: windows
The repo should consists of two projects:
- The unity project
- A app for players

## Unity project 
The unity project should act as a host and it should also be the general display for the game. It should work like a server that recieves player instructions and communicates back to the player app.

## The app
The app should be a small ui, where you connect to the host via a code. Here we can choose add some genreal settings like tank color, standard script to deploy at first round.

We will get info on how much time have elapsed in the round, how much health our tank has
